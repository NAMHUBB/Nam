# DNN(Deep Neural Network, 심층 신경망)

인간의 뇌 구조를 모방한 인공 신경망(ANN)에서 은닉층(Hidden Layer)을 여러 개 쌓아 올린 형태

## 1. DNN의 기본 구조
DNN은 크게 세 가지 종류의 층(Layer)으로 구성됩니다.

- 입력층 (Input Layer): 외부로부터 데이터를 받아들이는 층입니다. (예: 이미지의 픽셀 값, 주가 데이터 등)

- 은닉층 (Hidden Layer): 입력된 데이터에서 특징(Feature)을 추출하고 복잡한 연산을 수행합니다. 이 층이 2개 이상일 때 '심층(Deep)' 신경망이라고 부릅니다.

- 출력층 (Output Layer): 최종적인 예측값이나 분류 결과를 내놓는 층입니다.

## 2. 핵심 동작 원리
DNN이 학습하는 과정은 크게 두 단계로 나뉩니다.

- 순전파 (Forward Propagation): 데이터를 입력받아 각 층의 가중치(W)와 편향(b)을 곱하고 더해 결과를 산출합니다. 이때 **활성화 함수(Activation Function)**를 거쳐 비선형성을 추가합니다.

- 역전파 (Backpropagation): 출력값과 실제 정답 사이의 오차(Loss)를 계산한 뒤, 이 오차를 줄이기 위해 **경사 하강법(Gradient Descent)**을 사용하여 가중치를 거꾸로 거슬러 올라가며 업데이트합니다.

## 3. 왜 'Deep'하게 쌓을까?
층이 깊어질수록 신경망은 더 추상적인 특징을 학습할 수 있습니다.

- 낮은 층: 선, 점, 면 같은 단순한 형태를 인식.

- 높은 층: 눈, 코, 입 또는 복잡한 패턴과 사물의 형태를 인식.

## 4. 주요 활용 분야
DNN은 현대 AI의 근간이 되는 기술로, 데이터의 특성에 따라 다양한 형태로 변형되어 사용됩니다.


- CNN	(합성곱 신경망):	이미지 인식, 자율 주행, 사물 검출
- RNN	(순환 신경망):	자연어 처리, 번역, 시계열 데이터 분석
- GAN	(생성적 적대 신경망):	이미지 생성, 딥페이크, 디자인 자동화
