#  echo print printf 차이

| 명령어      | 사용하는 곳             | 특징                  |
| -------- | ------------------ | ------------------- |
| `echo`   | **쉘(Bash)**        | 가장 간단한 출력 도구        |
| `printf` | **쉘(Bash)**        | **형식 지정 가능**한 출력 도구 |
| `print`  | **Python** 등 일부 언어 | Bash에서는 **사용하지 않음** |  
<br>


#  목적
| 목적           | 추천 명령어   | 이유             |
| ------------ | -------- | -------------- |
| 간단한 출력       | `echo`   | 빠르고 간편         |
| 줄 맞춤/숫자 포맷   | `printf` | 정밀 제어 가능       |
| Python 코드 작성 | `print`  | Bash에서는 사용 안 됨 |
