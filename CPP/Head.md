# CPP HEAD

### ✅ 1. 입출력 관련(I/O)

| 헤더           | 설명                         |
| ------------ | -------------------------- |
| `<iostream>` | 기본 입출력 (cout, cin, cerr)   |
| `<iomanip>`  | 출력 형식 제어 (setw, setfill 등) |
| `<fstream>`  | 파일 입출력                     |
| `<sstream>`  | 문자열 스트림 (문자열을 스트림처럼 처리)    |


### ✅ 2. 문자열 관련
| 헤더          | 설명                              |
| ----------- | ------------------------------- |
| `<string>`  | std::string 사용                  |
| `<cstring>` | C 스타일 문자열 함수 (strlen, strcpy 등) |
| `<cctype>`  | 문자 검사 함수 (isdigit, isalpha 등)   |


### ✅ 3. 수학/숫자 관련
| 헤더          | 설명                       |
| ----------- | ------------------------ |
| `<cmath>`   | 수학 함수 (sin, cos, sqrt 등) |
| `<cstdlib>` | rand, atoi 등 C 표준 라이브러리  |
| `<limits>`  | 자료형의 한계치 확인              |
| `<climits>` | 정수형의 최소/최대값              |


### ✅ 4. 시간 관련
| 헤더         | 설명                 |
| ---------- | ------------------ |
| `<ctime>`  | 시간/날짜 함수           |
| `<chrono>` | 고정밀도 시간 측정(현대 C++) |


### ✅ 5. 컨테이너 (STL)
📌 배열/벡터/리스트
| 헤더               | 설명        |
| ---------------- | --------- |
| `<vector>`       | 동적 배열     |

| `<forward_list>` | 단일 연결 리스트 |

📌 스택/큐/덱
| 헤더        | 설명 |
| --------- | -- |
| `<stack>` | 스택 |
| `<queue>` | 큐  |
| `<deque>` | 덱  |

📌 집합/맵
| 헤더                | 설명        |
| ----------------- | --------- |
| `<map>`           | 키-값 저장    |
| `<unordered_map>` | 해시 기반 map |
| `<set>`           | 정렬된 집합    |
| `<unordered_set>` | 해시 기반 set |


### ✅ 6. 알고리즘 / 유틸리티
