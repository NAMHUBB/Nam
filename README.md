**### NAM's study

---

#### Computing Foundation
- Processing Unit
  - CPU: 명령어 집합(ISA), 마이크로아키텍처
  - GPU/NPU: 병렬 처리, 가속기 구조
- Computer Architecture
  - Memory Hierarchy: 캐시, RAM, SSD의 속도 차이와 관리
  - Bus System: 구성 요소 간 데이터 전송 통로
    
#### Operating System
- OS
  - Kernel: 프로세스 관리, 스레드, CPU 스케줄링
  - Memory Management: 가상 메모리, 할당 및 해제 정책
  - File System: 데이터의 물리적 저장 및 관리 방식
    
#### Algorithms & Data Structures
- Algorithms
  - 정렬 및 탐색: Quick Sort, Binary Search 등
  - 문제 해결 전략: Dynamic Programming, Greedy, Recursion
- Data Structures
  - 선형 구조: Array, Linked List, Stack, Queue
  - 비선형 구조: Tree, Graph, Hash Table (알고리즘의 재료)
    
#### Programming Languages & Dev
- Programming Languages
  - 컴파일러/인터프리터: 소스코드가 기계어로 변환되는 과정
  - 프로그래밍 패러다임: 객체지향(OOP), 함수형 프로그래밍
- Software Engineering
  - Design Patterns: 코드의 구조적 설계 방법
  - Version Control: Git, GitHub를 통한 협업
    
#### Network & Connectivity
- Network
  - OSI 7 Layer & TCP/IP: 통신 계층의 표준 모델
  - IP: 주소 할당, 서브네팅, 라우팅 원리
- Communication Protocols
  - HTTP/HTTPS: 웹 통신 규약
  - Web Socket: 실시간 양방향 통신
        
#### Web Infrastructure
- Server
  - Web Server: Nginx, Apache 등 정적 파일 처리
  - WAS (Web Application Server): Tomcat, Node.js 등 로직 처리
- Domain
  - DNS (Domain Name System): 도메인 주소를 IP로 변환하는 과정
  - SSL Certificates: 도메인 보안 인증
      
#### Database
- DB
  - RDBMS: SQL, 관계 모델, 데이터 무결성(ACID)
  - NoSQL: 대용량 데이터 분산 저장(Redis, MongoDB)
- Data Modeling
  - 스키마 설계, 정규화(Normalization), 인덱싱(Indexing)
        
#### Cloud & DevOps
- Cloud
  - Cloud Service: AWS, Azure, GCP 기반 자원 관리
  - Infrastructure as Code (IaC): 코드로 관리하는 인프라
- Virtualization
  - Container: Docker를 이용한 환경 격리
  - Orchestration: Kubernetes를 통한 대규모 서버 관리

#### Cyber Security
- Cyber Security
  - Cryptography: 대칭키/공개키 암호화, 해시 함수
  - Network Security: 방화벽(Firewall), VPN, 침입 탐지 시스템(IDS)
  - Application Security: SQL Injection, XSS 공격 방어

#### Artificial Intelligence
- AI
  - Machine Learning: 데이터 기반 예측 및 분류 모델
  - Deep Learning: 신경망(Neural Networks), LLM(거대언어모델)
- Data Science
  - 데이터 전처리, 통계적 분석 및 시각화
