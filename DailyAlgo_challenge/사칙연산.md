###  두 수의 덧셈

- 문제 설명

두 개의 자연수 `number1`과 `number2`가 주어질 때, 이 두 수의 합을 반환하는 `solution` 함수를 작성하시오.

- 제한 사항

`number1`과 `number2`는 1 이상 100 이하의 자연수입니다.

<br>

```python
def solution(number1, number2)
  return number1 + number2
```
or
```Python
def solution(number1, number2)
  answer = number1 + number2
  return answer
```
